## Project Identifiers

The following are claimed project identifiers:

No DJKI (Direktorat Jenderal Kekayaan Intelektual) registration is claimed as
completed.

## License Separation

The software license (AGPL-3.0-only) and documentation license (CC-BY-SA-4.0)
do **not** grant unrestricted trademark rights. The following are reserved
separately:

- R-Think name
- R-Think™ wordmark
- R-Think logo
- Official certification claims
- Official compatibility claims
- Official endorsement claims
- Confusingly similar branding

## Permitted Use

- **Nominative factual reference:** You may refer to R-Think factually in
  descriptive contexts (e.g., "This project uses the R-Think framework")
- **Factual compatibility:** You may state factual compatibility without
  implying official endorsement

## Prohibited Use Without Permission

- **Implied official status:** Forks and derivatives must not imply official
  status, certification, sponsorship, or endorsement
- **Confusing branding:** Modified versions must use sufficiently distinct
  branding unless written permission is granted
- **Logo use:** Logo use requires explicit permission except for factual
  reference where legally allowed
- **® symbol:** Do not use ® unless trademark registration is completed and
  applicable in the relevant jurisdiction

## Modified Versions

If you fork or modify R-Think:

- Use sufficiently distinct branding to avoid confusion
- Do not imply your version is the official R-Think
- Do not use the R-Think™ wordmark in your project name without permission
- Retain attribution to the original project
